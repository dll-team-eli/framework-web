# 🏠 Project Eli — Smart Home Assessment Tool

A UX research and design project focused on helping families of adults with autism transition to semi-independent living through smart home technology and structured assessment tools.

---

## 📌 Overview

This project explores how smart home technology and structured assessment frameworks can support adults with autism in semi-independent living environments. The core deliverable is a **Home Environment Frustration Assessment Framework** — a tool caregivers and families can use to identify home-based challenges and prioritize solutions before any physical or technological retrofitting begins.

---

## 🎯 Problem

Families preparing adults with autism for semi-independent living face a fragmented landscape of smart home tools, safety modifications, and caregiving workflows. There is no structured, accessible way to identify the most critical risks in a home, prioritize what to address first, or connect those problems to existing solutions.

---

## 🧩 What We Built

- **Assessment Framework** — Room-by-room identification of challenges paired with recommended solutions
- **Home Assistant Integration Chart** — Smart home tools and their compatibility with the Home Assistant platform
- **AI Chatbot Guide** — Walks caregivers through the framework, answers questions, and surfaces dangerous scenarios
